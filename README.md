# AI Chatbot Mentor

An AI-powered Streamlit application that acts as a personalized learning mentor by answering questions based on the selected module and years of experience using LangChain and Google Gemini API.

## Features

Select learning module (Python, SQL, EDA, ML, etc.)

Choose years of experience for tailored responses

AI answers only module-specific questions

Maintains chat history during the session

Download chat conversation as a text file

Simple and interactive Streamlit UI

## Tech Stack

Python

Streamlit

LangChain

Google Gemini API
